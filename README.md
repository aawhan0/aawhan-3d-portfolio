# Aawhan 3D Portfolio

An interactive AI-powered portfolio built using the MERN stack and Three.js.

This project transforms a traditional portfolio into a physics-driven 3D experience with clean, modern UX and AI-enhanced project explanations.

---

## 🚀 Vision

To build a premium, interactive portfolio that combines:

- Clean minimal UI
- 3D physics-based interactions
- AI-powered project explanations
- Full-stack MERN architecture

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Three.js
- React Three Fiber
- TailwindCSS
- Framer Motion
- Zustand

### Backend
- Node.js
- Express
- MongoDB

### AI Layer (Planned)
- LLM integration for contextual project explanations
